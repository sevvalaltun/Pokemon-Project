# Pokemon Project (C)

This project is a simple Pokémon game developed using the C programming language.
It demonstrates modular programming, file handling, and basic game logic.

## Technologies
- C Language
- GCC / Clang

## Files
- main.c – Program entry point
- game.c / game.h – Game logic
- initialize.c / initialize.h – Initialization
- pokemon.h – Pokémon structures
